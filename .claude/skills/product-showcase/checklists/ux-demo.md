# Checklist UX Demo

## Pre-Publicacion

### Claridad del Valor

- [ ] Propuesta de valor visible en primeros 5 segundos
- [ ] Titulo descriptivo (no generico)
- [ ] Subtitulo que explica el "como"
- [ ] CTA principal obvio y contrastante

### Contenido Visual

- [ ] Before/after incluido y funcional
- [ ] Imagenes optimizadas (<200KB cada una)
- [ ] GIF/video carga rapidamente
- [ ] Graficos legibles en mobile
- [ ] Colores consistentes con la marca

### Metricas y Datos

- [ ] Numeros destacados visualmente
- [ ] Comparativas claras (mejor vs peor)
- [ ] Fuente de datos mencionada si aplica
- [ ] Sin decimales excesivos (81.9% no 81.91234%)

### Interactividad

- [ ] Slider before/after funciona en mobile
- [ ] Botones tienen estados hover/active
- [ ] Formularios validados correctamente
- [ ] Feedback visual en acciones

### Rendimiento

- [ ] Lighthouse Performance > 80
- [ ] First Contentful Paint < 2s
- [ ] Time to Interactive < 4s
- [ ] Sin errores en consola
- [ ] Imagenes lazy-loaded

### Responsive

- [ ] Probado en iPhone SE (320px)
- [ ] Probado en tablet (768px)
- [ ] Texto legible sin zoom
- [ ] Botones tocables (min 44px)
- [ ] Sin scroll horizontal

### Accesibilidad

- [ ] Contraste texto/fondo > 4.5:1
- [ ] Alt text en imagenes
- [ ] Formularios con labels
- [ ] Navegable con teclado
- [ ] Focus visible en elementos

### Links y CTAs

- [ ] Todos los links funcionan
- [ ] Links externos abren en nueva tab
- [ ] CTA principal destaca sobre secundarios
- [ ] Texto del boton describe la accion

## Post-Publicacion

### Analytics

- [ ] Google Analytics configurado
- [ ] Eventos de conversion trackeados
- [ ] Heatmaps instalados (opcional)
- [ ] Links con UTM parameters

### SEO

- [ ] Title tag unico y descriptivo
- [ ] Meta description < 160 chars
- [ ] Open Graph tags configurados
- [ ] Favicon incluido

### Social

- [ ] Preview de LinkedIn verificado
- [ ] Preview de Twitter verificado
- [ ] Imagen social 1200x630px
- [ ] Descripcion social atractiva

## Errores Comunes a Evitar

### Diseno

- No usar mas de 3 fuentes
- No usar mas de 5 colores
- No centrar texto largo
- No usar iconos sin contexto

### Contenido

- No escribir parrafos largos
- No usar jerga tecnica excesiva
- No olvidar el CTA
- No esconder las metricas importantes

### Tecnico

- No cargar imagenes de 5MB
- No olvidar favicon
- No dejar console.logs
- No hardcodear URLs
