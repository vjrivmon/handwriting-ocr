# Principio de Prägnanz (Buena Forma)

## Resumen

El cerebro prefiere interpretar estímulos ambiguos de la manera más simple
posible. Dados múltiples interpretaciones, se elige la más simple, simétrica y
ordenada.

## Origen

- **Escuela**: Psicología de la Gestalt
- **Año**: ~1920s
- **Término**: Del alemán "Prägnanz" (concisión, esencia)
- **Autores principales**: Max Wertheimer, Kurt Koffka, Wolfgang Köhler

## Fundamento Psicológico

El sistema visual tiene recursos limitados y busca eficiencia. Formas simples
requieren menos procesamiento cognitivo que formas complejas. Esta preferencia
por la simplicidad es la **ley fundamental** de la Gestalt de la cual derivan
los otros principios.

## Aplicación en Diseño

### Iconografía Simple

- Formas geométricas básicas
- Mínimo número de elementos
- Siluetas reconocibles
- Evitar detalles innecesarios

### Layouts Ordenados

- Grids regulares y predecibles
- Alineación consistente
- Proporciones armónicas
- Whitespace deliberado

### Interfaces Limpias

- Un elemento principal por pantalla
- Reducir opciones a lo esencial
- Eliminar decoración superflua
- Simplificar antes de agregar

### Información Visual

- Charts con mínimos elementos
- Data-ink ratio alto (Tufte)
- Labels solo donde necesarios
- Colores con propósito, no decoración

## Ejemplos

- **Apple**: Productos y UI minimalistas
- **Google Homepage**: Solo barra de búsqueda
- **iOS Icons**: Formas simples y reconocibles
- **Medium**: Diseño centrado en contenido
- **Notion**: Interface que desaparece ante el contenido

## Anti-patterns

- ❌ Iconos con demasiados detalles
- ❌ Layouts irregulares sin justificación
- ❌ Gradientes y sombras excesivos
- ❌ Múltiples elementos compitiendo por atención
- ❌ Información visual redundante

## Métricas

- **Visual Complexity Score**: Análisis de elementos por pantalla
- **Signal-to-Noise Ratio**: Contenido útil vs decorativo
- **Icon Recognition Speed**: Tiempo para identificar iconos
- **Simplicity Rating**: Evaluación subjetiva de usuarios
